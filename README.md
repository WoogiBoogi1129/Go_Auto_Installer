# Go_Auto_Installer
Go 언어 자동설치기 입니다.

Go Version: 1.21.10

## 사용방법
- git clone 해오기
```sh
git clone https://github.com/WoogiBoogi1129/Go_Auto_Installer
```
- shell 파일 실행 권한 부여
```sh
chmod +x GO_Auto_Installer/Go_Auto_Installer.sh
```

- shell 파일 실행
```sh
sudo ./Go_Auto_Installer/Go_Auto_Installer.sh
```
